## 🌐 Overview

CORTEX is not a regular typing test. It is a **fast reaction training game** built especially for people using split ergonomic keyboards.

Instead of typing in a simple, linear way, CORTEX trains you to:
- Use both hands independently  
- React quickly to words appearing on different sides of the screen  
- Improve accuracy while typing under time pressure  

It is built as a single HTML file that runs directly in your browser. No installation, no setup tools, and no extra downloads—just open and play.

---

## 🚀 Key Features

* **Left & Right Hand Training:** Words appear on the left or right side of the screen. You must type them using only the matching hand. This helps train split keyboard coordination.

* **Adaptive Difficulty:** The game automatically adjusts speed based on your performance. Typing accurately makes the game faster and more challenging. Mistakes slow it down slightly.

* **Short 45-Second Rounds:** Each round is short and intense. Survive the timer to move to the next level, where the difficulty increases.

* **Clean Visual Style:** A modern interface with smooth effects that works well on both large monitors and normal laptops.

* **Score Saving:** Your best scores are saved in your browser so you can track improvement over time.

* **Immersive Sound Feedback:** Typing actions include responsive sound effects that help you feel timing and rhythm.

### 📁 Custom Keyboard Setup

Upload a JSON layout file in the setup menu. CORTEX will:
- Read your layout  
- Map your keys correctly  
- Adjust gameplay to match your keyboard  

No manual setup needed.

---

## 🕹️ How to Play

1. Type the first letter to select a word  
2. Focus on one word at a time  
3. Avoid mistakes to keep your combo  
4. Fill your energy bar by typing accurately  
5. Trigger a bonus effect when full for extra points  

---

## ⚠️ Word Types

* **Standard:** Normal speed words  
* **Fast:** Very quick targets  
* **Moving:** Slightly shifting words  
* **Long:** More complex words requiring focus
